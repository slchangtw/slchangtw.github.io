How to use Pelican on Github Page

Published Mon 01 March 2021 in misc

by Shun-Lung Chang 

# in the terminal, create a virtual environment 
$ python -m venv .venv
# activate the virtual environment
$ source .venv/bin/activate
# install pelican
$ (.venv) pip install pelican ghp-import
# create a skeleton project
$ pelican-quickstart

The posts are located in the content folder of the skeleton project. The post can be in the format of either .rst or .md. Create a file example.md in this folder and the content is shown below.

Title: My title
Date: 2010-12-03 10:20
Modified: 2010-12-05 19:30
Category: Python
Authors: John Doe
Summary: Short version for index and feeds

This is the content of my blog post.

We can convert the content into HTML via the pelican command and launch a simple web server at http://localhost:8000/ to preview the site.

# convert the content into HTML
$ pelican content -o output -s pelicanconf.py
# launch a local server
$ cd output
$ python -m http.server

To publish the site on Github page, we have to create a repository <your username>.github.io on Github. With the help of the package ghp-import, we can easily push the content to Github by the running:

$ ghp-import output -b gh-pages
$ git push git@github.com:<your username>/<your username>.github.io.git gh-pages:master

In the end, the website can be seen at https://<your username>.github.io
