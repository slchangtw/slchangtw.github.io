How to use assertions in Regular Expression

Published Sun 14 March 2021 in Python

by Shun-Lung Chang Regex

An assertion in Regular Expression returns the result without consuming up the characters in the string. That is the reason it is called assertion, which returns match or no match.

The assertions can be positive lookahead and positive lookbehind.

Positive Lookahead

regex_1(?=regex_2)

The positive lookahead (?=) asserts that regex_1 is immediately followed by regex_2.

import re

# E.g., find the occurrence of "t" followed immediately by "o"
pattern = r"t(?=o)"

re.search(pattern, "potato")
# <_sre.SRE_Match object; span=(4, 5), match='t'>

Negative Lookahead

regex_1(?!regex_2)

The negative lookahead (?=) asserts that regex_1 is NOT immediately followed by regex_2

import re

# E.g., find the "p" that is not followed immediately by "p".

# \1 means the first captured group
# that is, a backreference which will be translated into the same character found in (p), that is, "p".
pattern = r"(p)(?!\1)"

re.search(pattern, "apple")
# <_sre.SRE_Match object; span=(2, 3), match='p'>

Positive Lookbehind

(?<=regex_2)regex_1

The positive lookbehind (?<=) asserts that regex_1 is immediately preceded by regex_2

import re

# E.g., find the digit which are immediately preceded by odd digit.
pattern = "(?<=[13579])[\d]"

re.search(pattern, "0123")
# <_sre.SRE_Match object; span=(2, 3), match='2'>

Negative Lookbehind

(?<!regex_2)regex_1

The positive lookbehind (?<!) asserts that regex_1 is NOT immediately preceded by regex_2

import re

# E.g., find the character which is not immediately preceded by vowels (a, e, i, u, o, A, E, I, O, U).
pattern = "^.(?<![aeiouAEIOU])."
# <_sre.SRE_Match object; span=(0, 1), match='a'>

Usage

Suppose we need find all cloud architects whose expertise is AWS from a list as follows. The key word "AWS" can be either before or after "Cloud Architect" but not in-between. We can use positive lookaheads to assert that the phrase matches "AWS" in either order, and the pattern is r'^(?=.*AWS).*Cloud Architect.*$'.

The .* in the lookahead (?=.*AWS) catches "Cloud Architect" and matches the condition "Cloud Architect", so "Cloud Architect AWS" can be found.

titles = ["AWS Cloud Architect", "Azure Cloud Architect", "Cloud Architect AWS", "Cloud AWS Architect", "AWS Cloud Sales"]

pattern = re.compile(r'^(?=.*AWS).*Cloud Architect.*$')

for title in titles: 
    print(pattern.search(title))
# <_sre.SRE_Match object; span=(0, 19), match='AWS Cloud Architect'>
# None
# <_sre.SRE_Match object; span=(0, 19), match='Cloud Architect AWS'>
# None
# None
